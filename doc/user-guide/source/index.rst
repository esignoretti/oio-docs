=================
OpenIO User Guide
=================

Abstract
~~~~~~~~

OpenIO SDS is an open-source object storage solution. It provides its own API and can be used through other APIs (like Openstack Swift or AWS S3) using related projects.  
This guide shows object storage users how to use our solution using compatible clients.  
It presents the usage of the OpenIO command line interface and simple usage of other clients, refer to their respective project page for an extensive documentation.
